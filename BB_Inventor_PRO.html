Sorgente dati

Scegli dove salvare e caricare il database.

I dati vengono salvati nel localStorage del browser / app. Nessuna configurazione necessaria.

✓ Sempre disponibile

Connetti a un server REST nella LAN o su internet (es. il mini-server Node.js incluso nel pacchetto Electron).

Carica il database da un URL pubblico (es. file JSON su GitHub, CDN, server web). Sola lettura all'avvio — i salvataggi restano in locale.

Salva e carica il database via FTP/FTPS. Richiede la versione Electron.

Salva il database come file JSON su Google Drive. Inserisci il tuo token OAuth2 (scope: drive.file).

Per ottenere il token: Google OAuth Playground → scope https://www.googleapis.com/auth/drive.file

Salva il database su Dropbox tramite API v2.

Crea un token su Dropbox App Console → crea app → genera access token.

Salva il database su OneDrive / SharePoint tramite Microsoft Graph API.

Token da Graph Explorer → scope Files.ReadWrite.

Leggi e scrivi il database su una cartella di rete Windows condivisa. Richiede la versione Electron con accesso al percorso UNC.

Esempio: \\\\192.168.1.5\\Documenti\\BBInventor\\database.json

Server web remoto
